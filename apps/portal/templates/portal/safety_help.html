{# Private browsing #}

{% trans "Use private browsing" %}

{% trans "Private browsing (also called incognito mode) stops the browser from saving your history." %}

{% trans "How to open private browsing" %}
  • Chrome: {% trans "Press Ctrl + Shift + N (or tap the three dots at the top right, then 'New incognito window')" %}
  • Safari (iPhone): {% trans "Tap the tabs button at the bottom right, then tap 'Private'" %}
  • Edge: {% trans "Press Ctrl + Shift + N (or tap the three dots, then 'New InPrivate window')" %}
  • Firefox: {% trans "Press Ctrl + Shift + P (or tap the three lines, then 'New Private Window')" %}

{% trans "When you close a private window, the browser won't remember what you looked at." %}

{# Clear history #}

{% trans "Clear your browser history" %}

{% trans "If you didn't use private browsing, you can delete your history afterwards." %}

{% trans "How to clear your history" %}
  • Chrome: {% trans "Press Ctrl + H, then click 'Clear browsing data'" %}
  • Safari (iPhone): {% trans "Go to Settings, then Safari, then 'Clear History and Website Data'" %}
  • Edge: {% trans "Press Ctrl + H, then click 'Clear browsing data'" %}
  • Firefox: {% trans "Press Ctrl + Shift + Delete" %}
{# Shared devices #}

{% trans "Using shared devices" %}

  • {% trans "Don't save your password in the browser if others use the same device." %}
